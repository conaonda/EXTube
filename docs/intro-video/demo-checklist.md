# 데모 녹화 체크리스트

## 사전 준비

- [ ] 로컬 또는 스테이징 환경에서 서비스 구동 확인
- [ ] 데모용 YouTube 영상 선정 (30초~1분, 카메라 이동이 충분한 영상)
  - 후보 1: 건물 외관 워킹 투어
  - 후보 2: 공원/정원 산책 영상
  - 후보 3: 실내 공간 투어
- [ ] 사전 복원 완료 확인 (데모 중 대기 시간 단축용)
- [ ] 브라우저 북마크바 / 알림 숨기기
- [ ] 화면 해상도 1920x1080 설정

## 녹화 도구

- **권장**: OBS Studio (무료, 크로스 플랫폼)
- **설정**:
  - 출력 해상도: 1920x1080
  - FPS: 30
  - 포맷: MP4 (H.264)
  - 오디오: 불필요 (나레이션은 후처리)

## 녹화 시나리오

### Scene 1: URL 입력 (약 15초)
1. EXTube 메인 페이지 표시
2. YouTube URL 붙여넣기
3. "복원 시작" 버튼 클릭

### Scene 2: 진행 과정 (약 15초)
1. 진행률 바 표시
2. 각 단계 전환 캡처 (다운로드 → 추출 → 복원)
3. **팁**: 사전 복원된 결과를 사용하되, 진행 UI는 별도 캡처

### Scene 3: 3D 결과물 탐색 (약 30초)
1. 완료 후 3D 뷰어 자동 전환
2. 마우스로 천천히 회전 (360도)
3. 줌 인/아웃으로 디테일 확인
4. 배경색 전환 (검정 → 흰색)

### Scene 4: 갤러리 탐색 (약 10초)
1. 갤러리 페이지로 이동
2. 다른 샘플 카드 클릭
3. 3D 뷰어에서 결과 확인

## 후처리

- [ ] 불필요한 대기 시간 컷 편집
- [ ] 마우스 커서 강조 효과 추가 (선택)
- [ ] 슬라이드 영상과 합치기
- [ ] 자막 추가 (선택)
- [ ] BGM 추가 — 저작권 프리 음원 (선택)
